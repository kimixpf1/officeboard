## 1. 简介

办公面板是一个集日程管理、会议安排、办文跟踪于一体的智能工作台。

- **桌面端**：浏览器打开完整功能（推荐 Chrome/Edge）
- **移动端**：浏览器 + 微信内均可使用，微信有独立轻量上传页

**核心能力：**

| 能力 | 说明 |
|------|------|
| 事项管理 | 待办、会议、办文三类，支持跨日期、周期性 |
| AI 录入 | 拍照/上传文件自动识别，自然语言一句话创建 |
| 三视图 | 日视图（看板）、周视图、月视图自由切换 |
| 智能通知 | 待办到期提醒、倒数日轮播、自定义闹钟 |
| 云端同步 | 多设备数据一致，自动备份防丢失 |
| 实用面板 | 倒数日、通讯录、快捷链接、工具箱、便签 |

## 3. 事项管理

### 3.1 三种事项类型

> 📸 **截图位置：screenshots/02-三种事项卡片.png** — 并排展示待办（红色优先级标记+截止时间）、会议（蓝色+地点参会人）、办文（绿色+文号进度）三张典型卡片

| 类型 | 图标 | 用途 | 特有字段 |
|------|------|------|---------|
| **待办** | ✅ | 个人任务 | 优先级（高/中/低）、截止时间 |
| **会议** | 📅 | 会议安排 | 时间、地点、参会人 |
| **办文** | 📄 | 文件办理 | 文号、来文单位、起止日期、进度 |

### 3.2 新建事项（四种方式）

**方式一：点击"新增"按钮**

| 平台 | 操作 |
|------|------|
| 桌面 | 点击各列底部"新增"按钮 → 弹出编辑表单 |
| 移动端 | 同上，表单全屏显示 |

**方式二：自然语言输入**

在各列底部的输入框直接打字，AI 自动识别并创建：

| 输入示例 | 识别结果 |
|---------|---------|
| `明天下午3点项目评审会 301会议室` | 会议：明天 15:00，地点 301 |
| `周五前完成季度报告 #高` | 待办：本周五截止，高优先级 |
| `下周一前处理XX文件 文号2026-001` | 办文：下周一截止，文号 2026-001 |

- **桌面**：输入框在各列底部，回车提交
- **移动端**：输入框在页面底部，精简为单行

**方式三：文件上传**（详见 [第4章](#4-ai-智能录入)）

**方式四：日历空白处右键**

| 平台 | 操作 |
|------|------|
| 桌面 | 在周/月视图空白日期格子右键 → 弹出类型选择（待办/会议/办文）→ 日期自动填入 |
| 移动端 | 暂不支持（移动端无日历右键） |

### 3.3 编辑与删除

| 操作 | 桌面 | 移动端 |
|------|------|--------|
| **编辑** | 点击事项卡片，弹出编辑表单 | 同桌面 |
| **完成标记** | 待办/办文卡片上点击勾选按钮 | 同桌面 |
| **删除** | 右键菜单 → 删除 | 长按卡片 → 删除 |

### 3.4 跨日期办文

办文可设置起止日期（如 5月10日–15日），在周/月视图中横跨多天显示。

编辑跨日期办文时，弹出作用范围选择：

| 范围 | 说明 | 适用场景 |
|------|------|---------|
| **仅当天** | 只修改选中这一天 | 某天进度有变化 |
| **当天及之后** | 从选中日开始往后全部修改 | 从今天起改为新状态 |
| **全部** | 起止日期内所有天统一修改 | 整体调整 |

> 📸 **截图位置：screenshots/03-跨日期办文.png** — 月视图中一个横跨5天的办文卡片，旁边弹出作用范围选择弹窗

### 3.5 周期性任务

待办可设为周期性重复，支持：

| 周期 | 示例 |
|------|------|
| 每天 | 每日站会 |
| 每周 | 每周五周报 |
| 每月 | 每月1号提交考勤 |

设置后自动按规则生成后续事项，修改时可选择"仅本次"或"全部后续"。

## 4. AI 智能录入

### 4.1 上传图片/PDF

> 📸 **截图位置：screenshots/04-AI识别预览.png** — 上传一份会议通知PDF后的三列预览弹窗，标注新增(绿)/合并(蓝)/跳过(灰)三组

| 平台 | 触发方式 |
|------|---------|
| 桌面 | ① 点击顶部 📎 按钮 ② 直接拖拽文件到页面 |
| 移动端 | 点击 📎 按钮 → 拍照或从相册选择 |

支持格式：**JPG、PNG、PDF**（微信内仅支持图片）

**识别流程：**
1. 选择文件 → 上传
2. AI 分析内容，提取会议/待办/办文
3. 弹窗预览，三组展示：
   - 🟢 **新增**：新识别的事项，将直接添加
   - 🔵 **合并**：与已有事项匹配，信息合并（如会议补充参会人）
   - ⚪ **跳过**：完全重复，自动跳过
4. 可逐条编辑、删除或手动调整分组
5. 确认 → 批量保存

### 4.2 微信独立上传页

手机微信打开 `wechat-upload.html`：

- 精简界面，只有上传识别功能
- 拍照或选文件 → AI 识别 → 确认 → 自动保存
- 保存后 1 秒自动返回微信

## 5. 通知栏

通知栏位于页面顶部中央（280px 宽），是信息提醒的核心入口。

> 📸 **截图位置：screenshots/05-通知栏四种状态.png** — 四张并排截图：待办提醒(红色闪烁)、闹钟(橙色)、倒数日(黄色)、空闲(蓝色+宠物)

### 5.1 状态与颜色

| 优先级 | 状态 | 颜色 | 触发条件 |
|--------|------|------|---------|
| **1（最高）** | 待办到期提醒 | 🔴 红色闪烁 | 待办接近截止时间（默认提前3分钟） |
| **2** | 自定义闹钟 | 🟠 橙色 | 闹钟到达设定时间 |
| **3** | 倒数日轮播 | 🟡 黄色 | 未来 10 天内有倒数日/纪念日 |
| **4（默认）** | 空闲态 | 🔵 蓝紫 | 以上都不满足时显示 |

优先级高的会覆盖低的，例如待办提醒触发时会覆盖倒数日显示。

### 5.2 待办提醒

- 待办到期前通知栏变**红色闪烁**，可自定义提前多久提醒
- 新建/编辑待办时可设置**提前提醒**：3分钟 / 10分钟 / 30分钟 / 1小时 / 2小时 / 半天 / 1天 / 2天 / 3天
- 当提前量 ≥ 1天时，可指定**具体提醒时刻**（如提前1天 09:00 提醒，默认 09:00）
- 不设时刻则精确按提前分钟数提醒（如提前 30 分钟 = 截止前 30 分钟开始闪烁）
- 点击通知栏上的 **✓ 按钮**可直接完成该待办
- 多个待办同时到期时自动轮播（每 3 秒切换）
- 所有待办处理完毕或手动完成后，恢复下一级状态

### 5.3 自定义闹钟

| 平台 | 打开闹钟设置 |
|------|------------|
| 桌面 | 在空闲态通知栏上**右键** → 弹出闹钟管理窗口 |
| 移动端 | 在空闲态通知栏上**长按 0.6 秒** → 弹出闹钟管理窗口 |

**闹钟支持三种模式：**

| 模式 | 说明 |
|------|------|
| 每天 | 每日固定时间提醒 |
| 工作日 | 周一至周五提醒 |
| 指定星期 | 选择具体哪几天（如周一、周三、周五） |

闹钟触发时通知栏变橙色，提前 3 分钟开始闪烁。

### 5.4 空闲态：语录 + 宠物

没有提醒时，通知栏展示鸡汤语录和互动宠物。

| 操作 | 桌面 | 移动端 |
|------|------|--------|
| 切换语录/宠物 | 点击通知栏 | 点击通知栏 |
| 打开设置弹窗 | 点击通知栏 | 点击通知栏 |
| 与宠物互动 | 点击下方 🍖喂食 🚰喝水 🦮遛弯 🍪零食 | 同桌面 |

**7 只宠物：**

| 名字 | 物种 | 特征 |

| 点点 | 🐶 泰迪犬 | 圆耳朵、蓬松卷毛、摇尾巴 |
| 旺财 | 🐶 柴犬 | 垂耳、腮帮毛、摇尾巴 |
| 小橘 | 🐱 橘猫 | 三角尖耳、卷尾巴、胡须 |
| 滚滚 | 🐼 熊猫 | 圆耳、黑眼圈、胖身体 |
| 小灵 | 🦊 狐狸 | 大三角耳、蓬松尾巴、尖脸 |
| 团团 | 🐰 兔子 | 超长耳朵、棉花球尾巴 |
| 波波 | 🐧 企鹅 | 鸟喙、鳍状肢、白肚皮 |

## 11. 云端同步

登录 Supabase 账号后自动启用，右上角头像显示登录状态：
- 🟢 绿色边框 = 已登录，同步正常
- 🔴 红色/灰色 = 未登录或同步异常

**同步机制：**
- 数据变更后自动上传（3 次重试）
- 打开页面时自动拉取最新数据
- 多设备数据自动对账合并
- Side data（面板、主题、闹钟等）也纳入同步

**首次使用需配置 API Key：** 设置 → 填入 Kimi Key + Supabase Key。

## 12. 数据备份

系统自动多重备份，防止数据丢失：

| 备份方式 | 频率 | 保留份数 |
|---------|------|---------|
| 本地自动备份 | 同步前 | 最近 10 份（localStorage） |
| 云端每日备份 | 每晚 20:00 | 滚动 30 份（Supabase） |
| 本地文件下载 | 每晚 20:00 | 每份为独立 JSON 文件 |
| 手动导出 | 随时 | 下载 JSON 到本地 |

⚠️ **导入备份时自动暂停同步**，导入完成后恢复，避免导入数据被云端覆盖。
